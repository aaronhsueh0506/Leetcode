## Problem Description

Given `n` pairs of parentheses, write a function to generate all combinations of well-formed parentheses.

#### Example 1:
```plaintext
Input: n = 3
Output: ["((()))","(()())","(())()","()(())","()()()"]
```
#### Example 2:
```plaintext
Input: n = 1
Output: ["()"]
```
## Constraints

- `1 <= n <= 8`

## Concept
1. recursive for left and right count
